# Kalender

Workspace for creating and solving advent calendar coding puzzles.

## Usage

* npm install
* npm start <day>

If day doesn't exist, a new day will be created.

If done properly, output should be something like 
```
C:\code\kalender\days\2\index.js
0: passed
1: passed
2: passed
3: passed
Time: 0.305ms
Solution: <solution>
```
